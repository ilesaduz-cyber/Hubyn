# Hubyn ChatBot Lite

## Descrição

Motor de chatbot baseado em menus para WhatsApp.

## Objetivo

Fornecer respostas rápidas e automatizadas sem utilização de IA.

## Tecnologias

- Evolution API
- n8n
- PostgreSQL
- Docker

## Arquitetura

↓

↓
